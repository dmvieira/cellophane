Overview

Often when creating web applications, the developer must work on both client and server-side code. This tends to increase the amount of complexity and the number of context switches the developer must deal with.

One way to reduce this complexity is to make one end of the communications static and develop all logic on the other end. Cellophane is one attempt at implementing this paradigm, in the narrow scope of a 'web terminal' - a text input/output application that runs in the browser. Cellophane implements all of the client side code, rendering it static, and provides a very simple python library for interfacing with it over websockets. This allows the developer to focus all their attention on the server side, never having to consider any code or logic on the client side.

This python:

import cellophane

class Echoer(cellophane.Handler):
    def on_receive(self, message):
        self.writeline(message)

cp = cellophane.Cellophane(Echoer)
cp.go()
Try it out: click on the bottom bar, type a message, and hit enter.

Example: An anonymous chat server

import cellophane

clients = []

class BigTalker(cellophane.Handler):

    def on_create(self):
        clients.append(self)

    def on_destroy(self):
        clients.remove(self)

    def on_receive(self, message):
        message = cellophane.escape(message)
        self.writeline('You say, "%s"' % message, 'orange')
        for client in clients:
            if client is not self:
                client.writeline('Someone says, "%s"' % message)

cp = cellophane.Cellophane(BigTalker)
cp.go()
Try it out: click on the bottom bar, type a few words, and hit enter. Note: live demo includes code for rate limiting and blocking not in the above example.

Miscellaneous features:

  • Set colors for:
    • input and output backgrounds
    • output text on a per-line or character basis
    • input text
    • sepatator
  • Password mode for the input field.
  • Set your own favicon and page title.
  • Mechanism for periodic function calls outside of the scope of the Handler object.
